# 💼 Propuesta Comercial - Sistema de Subastas Ganaderas

## Para Empresas de Subastas en Casanare

---

## 🎯 Resumen Ejecutivo

El Sistema de Subastas Ganaderas es una plataforma web híbrida que digitaliza y moderniza el proceso tradicional de subastas, permitiendo participación presencial y remota simultánea, generando nuevas fuentes de ingreso y expandiendo el alcance de mercado.

---

## 💰 Modelo de Ingresos

### Comisiones por Transacción

**Estructura Estándar:**
- Comisión Compradores: **3%** del valor de compra
- Comisión Vendedores: **5%** del valor de venta
- **Total: 8%** del valor transado

### Ejemplo Práctico:

```
Lote vendido: $20,000,000

Comisión comprador (3%):  $   600,000
Comisión vendedor (5%):   $ 1,000,000
                          ___________
INGRESO TOTAL:            $ 1,600,000
```

